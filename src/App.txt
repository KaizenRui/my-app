START

Import dependencies:
  → import useState from 'react'          ← to manage state
  → import TaskForm from './TaskForm'     ← custom form component
  → import TaskList from './TaskList'     ← custom list display component

Initialize app:
  → Create state variable: tasks = []
  → Create state variable: filter = "all"
  → Create state variable: newDeadline = ""

App renders:
  → Call <TaskForm /> and pass down props:
      - addTask function
      - deadline value
      - setDeadline function

User types task text
  → TaskForm tracks value with state
  → Passed text is used by addTask()

User types deadline
  → newDeadline state is updated using setDeadline()

User clicks "Add" button
  → TaskForm calls addTask(text)
    → addTask():
        - Create new task object with id, text, completed=false, deadline
        - Add task to tasks[] using setTasks()
        - Clear newDeadline

App renders:
  → Show filter buttons ("All", "Open", "Completed")
  → Each button uses setFilter() to change the current view

User clicks a filter button
  → setFilter("all" | "open" | "completed")
  → filteredTasks = tasks filtered based on filter:
      - If filter == "all" → return all tasks
      - If filter == "open" → return tasks where completed == false
      - If filter == "completed" → return tasks where completed == true

App renders:
  → Call <TaskList /> and pass down props:
      - filteredTasks
      - deleteTask function
      - toggleTask function

User sees task list
  → Each task is displayed with:
      - text
      - deadline
      - checkbox (completion)
      - delete button (X)

User clicks checkbox on a task
  → toggleTask(id)
    → Find task by id
    → Flip task.completed true/false using setTasks()

User clicks delete button (X)
  → deleteTask(id)
    → Remove task by id using setTasks()

END
